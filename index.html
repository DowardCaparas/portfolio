About me

Hi, I'm Dounhuward Caparas, a front-end web developer from the Philippines. I build responsive web apps using ReactJS and Tailwind CSS. I'm currently sharpening my skills in front-end development with NEXT.js and TypeScript, and I've been exploring UI libraries to improve efficiency and responsiveness. While I'm focused on front-end work right now, I'm also looking forward to learning more about back-end development in the future.


I built this portfolio without any frameworks or libraries for now, keeping it simple with just HTML, CSS, and JavaScript to avoid over-engineering. The goal is to showcase my projects and journey. In the future, if I add a blog, extra pages, or features that are harder to implement without frameworks, I'll switch to using frameworks to speed up development and make it easier to create reusable components.


My tech stack includes Next.js, React, JavaScript, TypeScript, HTML, CSS, jQuery, Sass, and PHP. I also built a capstone project using C# and have a certificate for it. While I know how to use Bootstrap, I specialize in using Tailwind CSS. I have basic knowledge of Java, C++, and PHP.
